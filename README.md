## 🚀 Technical Stack

This website is built using:
- **[Quartz 4](https://quartz.jzhao.xyz/)**: A fast, battery-included static site generator.
- **Node.js**: Backend runtime for building the site.
- **GitHub Actions**: Automated CI/CD for seamless deployment.
- **Obsidian**: Used as the primary authoring tool for content.

---

## 📂 Project Structure

- `quartz/`: Core Quartz configuration and logic.
- `quartz/content/`: The "Brain" of the site. All markdown files, research notes, and publications live here.
- `quartz/static/`: Static assets like images and PDFs (including [Dr. Kumar's CV](https://ritesh001.github.io/ai4en/Ritesh_Kumar_CV.pdf)).

---

## 🛠️ Local Development

To run the site locally for testing or development:

1. **Install Dependencies**:
   ```bash
   cd quartz
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npx quartz build --serve
   ```
   The site will be available at `http://localhost:8080`.
